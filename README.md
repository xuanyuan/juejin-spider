# juejin-spider
本菜虫使用nodejs技术，利用Promise控制同步，比较蛋疼哈，异步的语言硬生生搞成同步。

1. 开发环境
* node v10.4.0
* ubuntu 18.04 LTS
* mysql 5.7.22-0ubuntu18.04.1
* redis-server v4.0.9

2. 准备工作和运行
* npm install

#开发调试环境
* node article.js

#生产环境
* pm2 start article.js
